# Mobile Wallet (Expo)

React Native client built with Expo.

## Requirements

- Node.js
- npm
- Expo CLI (`npm install -g expo-cli`)

## Quick Start

```bash
npm install
npm start
```

## Configuration

Copy `.env.example` to `.env` and set the API URL:

```bash
EXPO_PUBLIC_API_URL=http://localhost:12001
```

## Scripts

```bash
npm run ios
npm run android
npm run web
npm run lint
npm run typecheck
npm test
```
