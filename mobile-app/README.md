# Mobile App

React Native wallet application.

## Requirements

- Node.js
- npm
- React Native tooling (Xcode / Android Studio as needed)

## Setup

```bash
cd mobile-app
npm install
```

## Run

```bash
npm start
npm run ios
npm run android
```

## Configuration

Set API endpoints in your environment file (see `.env.example` if present):

```bash
API_ENDPOINT=http://localhost:12001
WS_ENDPOINT=ws://localhost:12001
```
