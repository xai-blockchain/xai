# External Audit Requirements

No external audit has been completed for this repository.

When commissioning an audit, require the following:
- Scope covering consensus, transaction validation, UTXO, P2P networking,
  wallet/CLI, staking, and governance modules.
- Threat model and explicit assumptions.
- Findings with severity, reproduction steps, and recommended remediations.
- Verification of fixes and retesting evidence.

Coordinate audits through security@xai.network.
