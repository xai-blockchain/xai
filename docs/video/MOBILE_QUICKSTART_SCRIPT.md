# XAI Mobile Wallet - Quick Start Video Script

Duration: 3-5 minutes
Target Audience: First-time blockchain users

## Scene 1: Introduction (15 sec)
- XAI logo animation with tagline
- "Get started with XAI in under 5 minutes"
- Visual: Clean, modern interface preview

## Scene 2: Download & Install (30 sec)
- Show App Store / Google Play Store
- Search for "XAI Wallet"
- Download XAI Wallet app
- Open app for first time
- Accept terms and conditions
- Grant necessary permissions

## Scene 3: Create Wallet (45 sec)
- Tap "Create New Wallet"
- Choose wallet name
- Set up biometric authentication (Face ID/Touch ID demo)
- CRITICAL: Write down 12-word recovery phrase
- Store recovery phrase securely offline
- Verify recovery phrase (select words in order)
- Wallet created successfully

## Scene 4: Get Testnet Tokens (30 sec)
- Navigate to "Receive" screen
- Copy wallet address or show QR code
- Open testnet faucet (in-app or browser)
- Paste address into faucet
- Request tokens
- Show push notification: "Tokens received"
- Balance updates in app

## Scene 5: Send Transaction (45 sec)
- Tap "Send" button
- Option 1: Scan recipient QR code
- Option 2: Paste recipient address
- Enter amount to send
- Review transaction details
- Confirm with biometric authentication
- Transaction submitted
- Show confirmation screen with transaction hash
- Link to block explorer

## Scene 6: View History (20 sec)
- Navigate to transaction history
- Show list of transactions (pending/confirmed)
- Tap transaction for full details
- View timestamp, amount, fees, confirmations
- Tap block explorer link for on-chain verification

## Scene 7: Security Tips (15 sec)
- Never share recovery phrase
- Enable biometric lock
- Verify addresses before sending
- Start with small test transactions

## Scene 8: Outro (15 sec)
- "You're ready to use XAI!"
- Links to documentation in the docs/ directory
- Community channels coming soon
- "Start building with XAI blockchain"

## Total Runtime: ~3:35 minutes

## Production Notes
- Use screen recording from actual app
- Add cursor highlights for tap actions
- Overlay text callouts for important steps
- Background music: Upbeat, tech-focused
- Voiceover: Clear, friendly, conversational tone
- Captions: Full transcription for accessibility
