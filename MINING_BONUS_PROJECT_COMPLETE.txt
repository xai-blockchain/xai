API ENDPOINTS
================================================================================

1. POST /mining/register
   Register new miner, auto-award early adopter bonus
   Request: {"address": "AXN..."}
   Response: registration_number, early_adopter_bonus, bonus_tier

2. GET /mining/achievements/<address>
   Check and award achievements
   Query: ?blocks_mined=X&streak_days=Y
   Response: new_achievements, total_new_bonus

3. POST /mining/claim-bonus
   Claim social media bonus
   Request: {"address": "AXN...", "bonus_type": "tweet_verification"}
   Response: success, amount, claimed_at

4. POST /mining/referral/create
   Generate unique referral code
   Request: {"address": "AXN..."}
   Response: referral_code, bonus_per_referral

5. POST /mining/referral/use
   Register with referral code
   Request: {"new_address": "AXN...", "referral_code": "CODE"}
   Response: referrer_bonus, success status

6. GET /mining/user-bonuses/<address>
   Complete user bonus summary
   Response: miner_stats, achievements, referrals, bonuses, summary totals

7. GET /mining/leaderboard
   Top miners by bonus earned
   Query: ?limit=10
   Response: ranked_list with total_bonus, bonus_count

8. GET /mining/stats
   System-wide statistics
   Response: total_miners, total_bonuses, active_codes, bonus_structure
