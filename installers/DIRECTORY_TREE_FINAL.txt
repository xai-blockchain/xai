Installation Methods
====================

1. Windows GUI Installer
   - Download .exe
   - Run installer wizard
   - Select components
   - Install

2. Windows Silent Install
   - xai-blockchain-setup.exe /VERYSILENT

3. macOS Homebrew (recommended)
   - brew tap xai-blockchain/tap
   - brew install xai
   - brew services start xai

4. macOS DMG
   - Open .dmg
   - Drag to Applications
   - Launch from Applications

5. macOS PKG
   - sudo installer -pkg xai.pkg -target /

6. Linux APT (Debian/Ubuntu)
   - sudo dpkg -i xai-blockchain.deb
   - sudo systemctl start xai-node

7. Linux DNF/YUM (Fedora/RHEL)
   - sudo dnf install xai-blockchain.rpm
   - sudo systemctl start xai-node

8. Linux AppImage (universal)
   - chmod +x XAI_Blockchain.AppImage
   - ./XAI_Blockchain.AppImage

Build Commands
==============

Windows:
  > "C:\Program Files (x86)\Inno Setup 6\ISCC.exe" setup.iss
  > makensis xai-nsis.nsi

macOS:
  $ ./create-dmg.sh
  $ ./create-pkg.sh
  $ brew install --build-from-source ./xai.rb

Linux:
  $ dpkg-buildpackage -us -uc -b
  $ rpmbuild -ba xai.spec
  $ ./create-appimage.sh

All Platforms (from project root):
  $ make -f installers/Makefile all
