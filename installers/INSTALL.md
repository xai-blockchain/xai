# Installation

This repo supports local installs and optional installer scripts.

## Local (recommended for development)

```bash
python -m venv venv
source venv/bin/activate
pip install .
```

## Installer Scripts

From the repo root:

```bash
./installers/install-xai.sh     # Linux/macOS
./installers/install-xai.ps1    # Windows
./installers/docker-install.sh  # Docker
```

Review each script before running it.
