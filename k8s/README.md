# Kubernetes

This directory contains Kubernetes manifests for running the node and related services. Review and adapt these files for your environment before applying them.

Key files:
- `namespace.yaml`
- `configmap.yaml`
- `secret.yaml`
- `statefulset.yaml`
- `service.yaml`
- `ingress.yaml` (optional)

Apply with `kubectl apply -f <file>` as needed.
